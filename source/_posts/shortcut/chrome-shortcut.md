shortcut | description
---------|------------
ctrl + t | 新建标签页
Ctrl + n | 新建窗口
Ctrl + h | 历史记录
ctrl + j | 下载页
ctrl + w | 关闭当前标签页
ctrl + p | 打印当前页
Ctrl + Tab | 切换到下一个标签页
Ctrl + Shift + Tab | 切换到上一个标签页
ctrl + o | 在当前变迁也中打开文件
ctrl + shift + o | 书签管理
alt + right | 前进
alt + left | 后退
Esc | 停止网页加载
